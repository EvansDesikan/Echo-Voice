## Requirements Summary
- Client records 15+ min of voice before death → ElevenLabs clones voice
- Client completes 60-question OCEAN-based psychometric questionnaire
- Client submits personal phrases and memories → stored in vector DB
- Family accesses a voice chat UI post-death → talks to AI agent
- Agent sounds like client (voice clone) + responds like client (personality engine + RAG)
- All data GDPR-compliant, EU-hosted, fully encrypted
- Working pitch demo required: pre-loaded sample profile + live conversation
- German + English language support

## Architecture Summary
- FastAPI backend exposes REST + WebSocket endpoints
- Voice pipeline: Whisper STT → Claude API (personality + RAG) → ElevenLabs TTS
- ChromaDB stores memory embeddings for RAG retrieval
- PostgreSQL stores client profiles, questionnaire scores, phrase banks
- MinIO stores raw voice recordings and audio assets
- React frontend: two modes — onboarding portal (client) + interaction UI (family)
- Full system runs via Docker Compose for the pitch demo

## Blockers Raised
- **Safety & Compliance → All:** GDPR requires explicit, recorded consent before any voice data collection → **Resolution:** Consent module built as gate #1 in onboarding flow; no data stored without cryptographic consent record
- **AI/ML Engineer → CTO:** ElevenLabs Professional Voice Clone requires API key with billing → **Resolution:** Evans uses own ElevenLabs account for demo; key goes in `.env` file (not committed)
- **Research Engineer → PM:** Psychological questionnaire must not constitute medical diagnosis (German law) → **Resolution:** Questionnaire framed as "personality preference" assessment, not clinical; disclaimer added

## Decisions Made
- Use ElevenLabs for voice cloning (pitch quality > self-hosting cost)
- Use Claude API (claude-sonnet-4-6) as agent LLM — Evans already has access
- Personality system uses Big Five (OCEAN) model — well-researched, widely accepted
- All sensitive audio processed locally via Whisper — never sent to third-party STT
- Project name for pitch: "ECHO Voice" — directly extends danach.de's existing "ECHO" product branding
